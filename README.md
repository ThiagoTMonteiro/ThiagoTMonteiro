<h2 align="left">About Me</h2>

###

<p align="left">- Olá, sou Thiago Tavares, um desenvolvedor Fullstack com mais de 5 anos de experiência profissional na criação de soluções corporativas completas — do backend ao frontend — com foco em integrações, escalabilidade e alta performance.<br><br>Tenho domínio do ecossistema .NET, além de sólida experiência com ReactJS, Angular 10+, Node.js, Java com Spring Boot e desenvolvimento mobile com Android e React Native.
